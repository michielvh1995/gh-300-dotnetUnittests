# This is a codespace setup for the GH-300 Exercises
The goal of this codespace is that a trainee can just fork this repo, open a codespace and use it, without any additional steps.

## Setup
1. Fork the repository
2. When done forking open a codespace on main
3. Perform the steps in [this exercise](https://microsoftlearning.github.io/mslearn-github-copilot-dev/Instructions/Labs/LAB_AK_04_develop_unit_tests_xunit.html)
starting with step *Examine the approach to unit testing*
